## Key Features

- **Audit-Focused AI**: Trained understanding of IIA Standards, COSO, COBIT
- **Document Analysis**: PDF, Excel, Word support for evidence review
- **Workpaper Generation**: Structured documentation with proper format
- **Compliance Research**: Web search for standards and regulations
- **Session Management**: Separate contexts for different engagements

## Modes of Operation

### Chat Mode

Interactive conversation for:

- Research questions
- Control analysis
- Procedure guidance
- Finding documentation

### Workpaper Mode

Structured document generation:

- Testing workpapers
- Risk assessments
- Executive summaries
- Audit reports
